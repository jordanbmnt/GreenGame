# Instructions

The objective is to get all of the green objects without running out of time. Every time you get a green the character position will reset.

- [W] OR UpArrow -> Move Up
- [A] OR LeftArrow -> Move Left
- [S] OR DownArrow -> Move Down
- [D] OR RightArrow -> Move Right

# Special Instructions and Prerequisites when installing

## Prerequisites

- Node.js
- npm (Node Package Manager)

## Instructions

Download or clone the project from the source code repository.

1. Open a terminal or command prompt and navigate to the project directory.

2. Install the project dependencies by running the following command:
   `npm install`.

3. Start the development server by running the following command:
  - `npm start`

4. Navigate to http://localhost:3000/

5. Press the Start button and enjoy!
